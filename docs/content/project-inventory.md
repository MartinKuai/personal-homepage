# Project Inventory

记录未来可能展示的作品，按优先级排列。

> 早期静态 demo 项目（如 Kuse Presales Demo、AI Knowledge Base MVP 等）不在当前首页展示。
> 如后续需要，可单独归档为 Archive / Early Experiments。

## 项目字段说明

| 字段 | 说明 |
|------|------|
| 项目名称 | 项目叫什么 |
| 项目类型 | Digital Product / Tool / Experiment / Content |
| 当前状态 | 完成 / 进行中 / 计划中 / 搁置 |
| 是否公开展示 | 是 / 否 / 待定 |
| 是否适合放首页 | 是 / 否 / 待定 |
| 项目链接 | 线上地址 |
| GitHub 链接 | 代码仓库 |
| 一句话说明 | 用一句话说清楚这个项目 |
| 背景 | 为什么做这个 |
| 做这个项目的原因 | 更具体的动机 |
| 解决的问题 | 这个项目解决了什么 |
| 使用的工具或技术 | 用了什么 |
| 体现的能力 | 能看出你会什么 |
| 当前不足 | 还有什么没做好 |
| 后续计划 | 下一步打算做什么 |
| 展示优先级 | P0 / P1 / P2 / P3 |
