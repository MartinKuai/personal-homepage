## 背景

这个网站用于长期记录我的项目、笔记、方法整理和阶段性思考。

这个项目用于建立一个可以持续维护、逐步积累的内容空间。

## 目标

目标是先搭建一个稳定、轻量、容易维护的个人网站结构。

在这个基础上，后续可以逐步补充项目案例、写作内容、方法记录和阶段性总结。

## 当前范围

当前已经完成的部分包括：

- 首页基础结构
- Writing 列表页
- Writing 详情页
- Markdown frontmatter 规范
- draft / placeholder 过滤
- Project Case 内容集合
- Project 详情页动态路由
- 首页 Works 区域的项目接入

## 技术选择

项目使用 Astro、TypeScript、Tailwind CSS 和 Markdown / MDX。

当前优先选择静态、轻量、可维护的方式，不引入数据库、登录、CMS、复杂表单或 AI API。

## 已完成内容

目前已经完成 V0 首页和 Writing 内容系统，并新增了第一个 Project Case。

首页结构保持简单，内容展示以长期维护为主。

Writing 和 Project 都通过 Markdown 文件管理，方便后续持续补充。

## 当前限制

这个项目还处在早期阶段。

目前 Project Case 数量较少，内容结构仍在验证中。

首页 Works 区域也还在观察是否适合长期展示项目案例。

## 下一步

后续会继续补充真实项目案例，整理已有笔记和方法，并逐步完善内容结构。

如果某些内容不适合放在首页，也会保留在内部文档或单独页面中。
