## 背景

这个网站用于长期记录我的项目、笔记、方法整理和阶段性思考。

它的重点是建立一个可以持续维护、逐步积累的内容空间。

## 目标

目标是搭建一个稳定、轻量、容易维护的个人网站结构。

在这个基础上，后续可以逐步补充项目案例、写作内容、方法记录和阶段性总结。

## 内容结构

当前页面包含几个主要区域：

- 首页概览
- Writing 文章列表和详情页
- Projects 项目复盘列表和详情页
- Timeline 阶段性记录
- Focus Areas 内容入口

每个区域承担不同功能，但都围绕同一个目标：让内容更容易被理解和进入。

## 技术选择

项目使用 Astro、TypeScript、Tailwind CSS 和 Markdown / MDX。

当前实现保持静态、轻量和易维护，以 Markdown 内容结构为主，降低长期维护成本，也方便后续持续补充。

## 已完成内容

目前首页、Writing、Projects 和 Timeline 已经形成基础结构，可以支撑后续内容持续补充。

首页结构保持简单，内容展示以长期维护为主。

Writing 和 Projects 都通过 Markdown 文件管理，方便后续持续补充。

## 当前状态

这个页面会随着后续项目、笔记和方法整理继续更新。

后续会继续补充更多真实项目复盘，让 Works 区域逐步从站点说明转向长期内容沉淀。

## 后续方向

后续会继续补充真实项目案例，整理已有笔记和方法，并逐步完善内容结构。

不适合放在首页的内容，会保留在独立页面中。
