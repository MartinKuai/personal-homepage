# Acceptance Self Check

Before reporting completion, verify the work against ACCEPTANCE_CHECKLIST.md.

## Check Dimensions

Check:

1. Task completion
2. UI language consistency
3. Empty page risk
4. Button and navigation behavior
5. Form validation behavior
6. Build / lint / test status
7. Scope creep
8. Unnecessary dependency additions
9. Sensitive file access
10. Unresolved TODOs

## Required Output

Update docs/handoffs/claude-to-codex.md with:

# Claude to Codex Handoff

## Completed Tasks

## Files Changed

## Validation Commands Run

## Acceptance Checklist Status

## Known Issues

## Questions for Codex Review

## Decision Rule

If any critical checklist item fails, do not claim the task is complete.
